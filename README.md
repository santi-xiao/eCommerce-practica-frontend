# Práctica de React

Práctica de la segunda evaluación del Liceo La Paz de desarrolo en entorno cliente.

Para ejecutar el proyecto debes **iniciar el ecommerce-backend**.

## Instalar dependencias de react

Ejecuta en la carpeta del frontend  
`npm install`

Una vez corriendo el backend ejecutarás el frontend con  
`npm start`
